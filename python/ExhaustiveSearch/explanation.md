# 완전 탐색(exhaustive search)

    - 정답이 될 가능성이 있는 모든 후보(candidates)를 체계적(반복문, 재귀, 비트마스크 등)으로 확인하는 것. ex. TwoSome 문제

    - 반복문
    - 재귀
    - 비트마스크

    > 백트래킹
        - solution이 될 가능성이 없는 candidate는 더 이상 탐색하지 않고 candidate를 포기(backtrack) 하면서 탐색
